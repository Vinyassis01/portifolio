Quais são seus projetos?

  • BoiAPI - Rest_framework : Uma API rest com Django_Rest_framework para coleta, armazenagem e acesso aos valores das arrobas do boi nos estados do Brasil . Com endpoints para busca por animal, estado e dia ,configurados para autenticacao com JWT, para POST e DELETE, com um scraper para coleta e tratamento dos valores .
    Tecnologias usadas :
    • Django
    • Rest_framework
    • Python
    • Beautiful Soap
    • JWT - Json Web Token
    • JSON
    • GitHub
  • DaillyFlix - Django : Um diario pessoal no estilo NetFlix onde pode ver seus momentos especias em um formato de serie, filme, anotacoes de um diario, podendo adicionar fotos ,capitulos ,temporadas ou capitulos em uma interface simples e intuitiva .
    Tecnologias usadas :
    • django
    • Python
    • GitHub
  • Docker-Jupyter : Uma imagem Docker baseada no python3.12-slim configurada com os pacotes python para Machine Learning e uma IDE jupyter .
    Tecnologias usadas :
    • Python
    • Jupyter
    • Docker
    • Scikit-learn
    • TensorFlow
    • GitHub
  • STRIDE - Logs gemini API : Com base no modelo de vulnerabilidades STRIDE ,desenvolvi um sistema simples que conecta com a API do gemini para avaliar um codigo de entrada e atraves de uma UI simples definir o tipo de retorno e seu formato .
    Tecnologias usadas :
    • Gemini API
    • Flet para a UI
    • GitHub
  • ML - Recomendation : Um sistema de machine Learning com scikit-learn para classisficar imagens de um dataset em classes e a partir desta classificacao determinar qual a peca de roupa e apos recomendar imagens similares do proprio dataset .
    Tecnologias usadas:
    • Scikit-learn
    • Matplotlib
    • GitHub
